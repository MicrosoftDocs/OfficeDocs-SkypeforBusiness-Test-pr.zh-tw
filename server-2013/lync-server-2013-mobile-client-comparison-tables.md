# Lync Server 2013 行動用戶端比較表

 

_**上次修改主題的時間：** 2015-07-17_

以下表格依照下列類別，比較 Lync 2013 行動用戶端與 Lync 2013 桌面用戶端之間的特性與功能：

  - 登入、推入通知與一般功能

  - 增強顯示狀態

  - 連絡人和連絡人群組

  - 立即訊息 (IM)

  - Lync 對 Lync 音訊與視訊

  - 會議

  - 電話語音

  - 外部使用者

  - 封存和規範

這些表格說明 Lync Server 2013 內部部署中 Lync 使用者可使用的功能。除了表格註腳說明的內容，相同的功能也適用於 商務用 Skype Online 和 Microsoft Office 365 使用者。
