# 群組來電接聽的部署程序

 

_**上次修改主題的時間：** 2015-03-09_

本節將概要說明部署群組來電接聽所需的步驟。您必須先部署具備 企業語音 的 Enterprise Edition 或 Standard Edition，之後才能設定群組來電接聽。在部署 企業語音 時就會安裝並啟用群組來電接聽所需的元件。
