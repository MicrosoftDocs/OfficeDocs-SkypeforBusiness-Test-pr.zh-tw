# 在 Lync Server 2013 中為 E9-1-1 設定網站資訊

 

_**上次修改主題的時間：** 2012-10-03_

若要定義子網路的位置原則，您必須以任何順序執行下列動作：

  - 將位置原則套用至網站。

  - 將子網路新增至網站。

如需有關網路網台的詳細資訊，請參閱＜[Lync Server 2013 的 Lync Server 網站](lync-server-2013-sites.md)＞。
