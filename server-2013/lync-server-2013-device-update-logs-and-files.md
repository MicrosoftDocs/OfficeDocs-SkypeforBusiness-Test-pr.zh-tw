# Lync Server 2013 中的裝置更新記錄與檔案

 

_**上次修改主題的時間：** 2013-02-20_

裝置更新記錄檔包含重要資訊，可用來管理及疑難排解 裝置更新 Web 服務。您可以變更記錄內容，並移除裝置記錄檔及更新不想要或不需要的記錄檔。本節說明如何使用 Lync Server 控制台 或 Lync Server 管理命令介面 修改記錄設定、清除裝置更新記錄，或是從伺服器移除記錄檔。
