# 在 Lync Server 2013 中使用 Windows PowerShell Cmdlet 疑難排解常設聊天室伺服器設定

 

_**上次修改主題的時間：** 2012-10-06_

請使用下列因應措施，以 Windows PowerShell 命令列介面 Cmdlet 來解決 常設聊天室伺服器 設定問題。

  - 如果顯示「未提供主體」錯誤訊息，則想新增的主體無法使用。
    
    > [!TIP]
    > 因應措施 – 無因應措施。此錯誤訊息表示主體無法使用。
