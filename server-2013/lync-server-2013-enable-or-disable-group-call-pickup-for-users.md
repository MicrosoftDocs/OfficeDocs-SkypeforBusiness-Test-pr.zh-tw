# 啟用或停用使用者的群組來電接聽

 

_**上次修改主題的時間：** 2013-02-22_

在群組來電接聽範圍新增至通話駐留範圍表之後，您就可以依需求針對使用者啟用及停用群組來電接聽。本節說明如何使用 SEFAUtil Resource Kit 工具，針對使用者啟用及停用群組來電接聽。

> [!NOTE]  
> SEFAUtil 參數 /enablegrouppickup 會啟用群組來電接聽並指派群組號碼。
